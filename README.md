# Miniatures Contest Platform

Generated: 2025-12-14

## Local quickstart
### 1) Services
```bash
cd backend
docker-compose up -d
```

### 2) Backend
```bash
cd backend
bun install
bun run dev
```

### 3) Frontend
```bash
cd frontend
npm install
npm run dev
```
